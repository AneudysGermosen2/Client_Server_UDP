# Client_Server_UDP
UDP pinger

 Client  sends a  simple  ping  message  to  the  server and recieves  a  corresponding  pong  message  backfrom  the server. After this, the delay between when the client sent the ping message and received the pong message is caculated.  This  delay  is  called  the  Round  Trip  Time(RTT).
